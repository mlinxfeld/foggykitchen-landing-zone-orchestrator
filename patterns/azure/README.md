# Azure Patterns

This directory contains the shared **Azure landing zone orchestration patterns** used by FoggyKitchen Landing Zone Orchestrator.

---

## 🎯 Purpose

These patterns turn Azure-focused YAML payloads into reusable architecture implementations built on top of FoggyKitchen modules and selected AzureRM resources where needed.
