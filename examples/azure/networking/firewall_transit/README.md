# Azure Firewall Transit Examples

This directory groups **Azure firewall transit landing zone variants** built on the shared FoggyKitchen orchestration pattern.

---

## 🎯 Purpose

The goal of this section is to collect examples where Azure Firewall provides the transit and egress control plane:

- spoke-to-spoke inspection through the hub
- default internet egress through Azure Firewall
- UDR-driven next hop enforcement
- validation workloads in isolated spokes
