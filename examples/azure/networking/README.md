# Azure Networking Examples

This directory contains **Azure networking-oriented landing zone examples** built on top of shared FoggyKitchen orchestration patterns.

---

## 🎯 Purpose

The goal of this section is to make Azure network patterns easy to discover and compare:

- baseline hub-and-spoke
- hub-and-spoke router VM transit
- firewall transit
- private endpoint service exposure
