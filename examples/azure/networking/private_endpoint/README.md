# Azure Private Endpoint Examples

This directory groups **Azure private endpoint landing zone variants** built on the shared FoggyKitchen orchestration pattern.

---

## 🎯 Purpose

The goal of this section is to collect examples where private service consumption is a first-class concern:

- private endpoint placement
- private DNS integration
- workload access over private paths
- validation of service consumption from Azure compute
