# Azure Hub-and-Spoke Examples

This directory groups **Azure hub-and-spoke landing zone variants** built on the shared FoggyKitchen orchestration pattern.

---

## 🎯 Purpose

The goal of this section is to separate:

- foundational hub-and-spoke scaffolding
- hub-based transit routing with a router VM

This keeps `basic` intentionally small while giving routing its own explicit scenario.
