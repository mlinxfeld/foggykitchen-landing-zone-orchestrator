# OCI LPG Local Peering Examples

This directory contains **OCI local peering landing zone examples** built on top of the shared FoggyKitchen LPG orchestrator pattern.

---

## 🎯 Purpose

The goal of this section is to group payload variants that demonstrate:

- same-region VCN-to-VCN connectivity through LPG
- private-first workload placement
- simple app-to-data topology patterns
