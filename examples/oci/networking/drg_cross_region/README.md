# OCI DRG Cross-Region Examples

This directory contains **OCI DRG cross-region landing zone examples** built on top of the shared FoggyKitchen DRG remote peering orchestrator pattern.

---

## 🎯 Purpose

The goal of this section is to group payload variants that demonstrate:

- remote peering across OCI regions
- VCN-to-VCN connectivity through DRG and RPC
- explicit VCN-side and DRG-side routing
